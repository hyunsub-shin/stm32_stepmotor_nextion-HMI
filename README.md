MCU : STM32F103RG
- UART1 : debug
- UART2 : Nection LCD
- TIM3 : PWM(Accel, Decel)

Driver : DRV8880
Motor : 21H4AB-05-049
- 1.8 degree/step
- 360/1.8 = 200 step
- 360 degree = 200 step = 1 mm
- 1 step = 0.005 mm
